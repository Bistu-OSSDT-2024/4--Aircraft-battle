飞机大战是基于MetaGPT生成的一个小游戏，飞机大战小游戏是一款经典的射击类休闲游戏，玩家将扮演一名勇敢的飞行员，驾驶战机在广阔的天空中与敌机进行激烈的战斗。游戏以简洁的画面、丰富的关卡和刺激的射击体验为特点，带给玩家无尽的挑战和乐趣。
